# StigChecklistCLI
CLI to copy STIG Viewer Checklist from a checklist (source) to another (target).


# Schemas Supported
* STIGViewer 2.9
* STIGViewer 2.10

# USAGE
```
stigchecklist-cli.exe copy -s "C:\[PATH]\STIGViewer_Checklist.ckl" -t "C:\[PATH]\New_STIGViewer_Checklist.ckl"
```
